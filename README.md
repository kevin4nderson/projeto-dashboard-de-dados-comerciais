# projeto-dashboard-de-dados-comerciais
Dashboard de BI para Análise Comercial - Power BI Desenvolvimento de solução com quatro visões analíticas. Inclui modelagem e ETL via Power Query para consolidação de dados. Uso de DAX para cálculo de KPIs (Faturamento, Margem, Volume e Ticket Médio), gerando insights para metas e comportamento de clientes.
